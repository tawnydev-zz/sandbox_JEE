# sandbox_JEE
Projet de démonstration JEE sur les taglibs et JSP.
<h2>Pré-requis:</h2>
<ul>
<li>Tomcat 8</li>
<li>JDK 1.7.0_79</li>
</ul>
<h2>Installation:</h2>
<ul>
<li>Ajouter standard-1.1.2.jar dans le dossier lib de Tomcat</li>
<li>Ajouter jstl-1.2.jar dans le dossier lib de Tomcat</li>
<li>Déployer sur le Tomcat 8</li>
</ul>
